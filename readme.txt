INSTRUCTIONS:

TO USE CALCULATOR PROGRAM, YOU CAN SET THE LEVEL OF VERBOSITY WITH -d FOR DEBUG, -i FOR INFO, OR -e FOR ERROR. THIS IS TO BE ENTERED AS THE FIRST ARGUMENT.

AS THE SECOND ARGUMENT, ENTER A VALID ARITHMETIC EXPRESSION AS A STRING USING THE PROPER FORMATTING.(USE QUOTATIONS " " AROUND STRING)

BY DEFAULT, THE PROGRAM VERBOSITY LEVEL IS SET TO ALL. IF YOU JUST ENTER AN ARITHMETIC EXPRESSION AS A STRING AS THE FIRST ARGUMENT.

THIS PROGRAM SUPPORTS THE FOLLOWING ARITHMETIC FUNCTIONS:add, sub, mult, div and let
